Description¶

My ML model will be classifying if customer will be engaged or unengaged.
Due to the specific of available data I'm treating the last period of transactions (3 weeks), as a future period (my target), and previously available not overlapping period (previous 3 weeks) as a current period.

I'm not going to focus on feature engineering, rather prepare some basic features and try to build good quality ML model that I understand.

A comment on hyperparameter selection:
It's possible to use an automatic hyperparameter tuner like NNI or scikit-opt, but here is rather redundant and I've decied to use trial and errors method and set parameters from hand.
Belove some justification about parameters selections:

  • boosting_type - traditional Gradient Boosting Decision Tree,
  • feature_fraction - fraction of features that are randomly selected in each iteration, the lower the more it prevents from strongest features domination,
  • learning_rate - a learning rate, just try few to find out what works the best (typically between 0.1 and 0.001),
  • max_bin - max number of bins features values will be bucketed in, if small training is faster but less accurate,
  • max_depth - tree max depth, the higher, the more is model prone to overfit,
  • n_estimators - generally could be large - I'm using early stopping anyway,
  • reg_alpha and reg_lambda - regularization factor, when number of features are low it could promote the strongest features and under valuate weaker features,
  • subsample - make training faster by reducing the number of data in each iteration,

def calculate_AUC_prop(model, Xtrain, Xtest, ytrain, ytest):
    y_pred_train = model.predict_proba(Xtrain)[:,1]
    fpr_train, tpr_train, thresholds_train = metrics.roc_curve(ytrain, y_pred_train, pos_label=1)
    roc_auc_train = metrics.auc(fpr_train, tpr_train)
    print("Train AUC= %.3f"% roc_auc_train)
    
    y_pred_test = model.predict_proba(Xtest)[:,1]
    fpr_test, tpr_test, thresholds_test = metrics.roc_curve(ytest, y_pred_test, pos_label=1)
    roc_auc_test = metrics.auc(fpr_test, tpr_test)
    print("Test AUC= %.3f"% roc_auc_test)    

    fig, ax = plt.subplots(figsize=(7,7))
    lw = 2
    ax.plot(fpr_train, tpr_train,  color='darkorange',
             lw=lw, label='Train ROC curve (area = %0.3f)' % roc_auc_train)
    ax.plot(fpr_test, tpr_test,  color='darkblue',
             lw=lw, label='Test ROC curve (area = %0.3f)' % roc_auc_test)
    ax.plot([0, 1], [0, 1], color='navy', lw=lw, linestyle='--')
    plt.xlim([0.0, 1.0])
    plt.ylim([0.0, 1.05])
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    plt.title('Receiver operating characteristic')
    plt.legend(loc="lower right")
    plt.show
    
    return (roc_auc_train,roc_auc_test)
